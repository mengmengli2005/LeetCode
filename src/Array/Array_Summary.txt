1. ^ 位运算异或 (Exclusive or): ^或⊕ 表示按位异或，如果两个数字对应的位相同 则得到0，相反 则得到1；
    a ^ 0 = a;  a = 100110, 100110 ^ 000000 = 100110;
    a ^ a = 0;  a = 100110, 100110 ^ 111111 = 011001;
    a ^ b ^ a = a ^ a ^ b = 0 ^ b = b;

2. | 位运算或：
    有按位相加的功能: a = 100110,  b = 000001;   a | b = 100111;

3. 在处理 Integer／Long 类型的 【重复Duplication】问题时，考虑用【位运算bit + %重复次数取余】的方法；
